# Excelize v2.10.1

{{ book.info }}

* 即將釋出：2026年2月25日

## 版本说明

此版本中最显著的变化包括：

### 兼容性提示

* 移除了 3 个导出的错误变量：`ErrStreamSetColStyle`、`ErrStreamSetColWidth` 和 `ErrStreamSetPanes`

### 新增功能

* 新增 `ChartDataPoint` 资料类型
* 在 `ChartSeries` 资料类型中新增 `DataPoint` 字段
* 在 `ChartAxis` 资料类型中新增 `DropLines` 和 `HighLowLines` 字段
* 在 `GraphicOptions` 资料类型中新增 `Name` 字段
* 在 `ChartSeries` 资料类型中新增 `DataPoint` 字段
* 新增 2 个常量：`MaxGraphicAltTextLength` 和 `MaxGraphicNameLength`
* 新增 7 个导出的错误变量：`ErrFillType`、`ErrFillGradientColor`、`ErrFillGradientShading`、`ErrFillPatternColor`、`ErrFillPattern`、`ErrMaxGraphicAltTextLength` 和 `ErrMaxGraphicNameLength`
* 新增 `GetHyperLinkCells` 函式，支持获取包含超链接的存储格，相关 issue [#1607](https://github.com/xuri/excelize/issues/1607)
* 新增 `GetSheetProtection` 函式，支持获取工作表保护設定
* 当向已存在批注的存储格添加批注时，`AddComment` 函式将返回错误
* 支持插入 ICO 格式图片，相关 issue [#2234](https://github.com/xuri/excelize/issues/2234)
* 新增 2 项公式函式：SORTBY 和 UNIQUE
* 添加图表时支持为圆环图、饼图和三维饼图設定资料点色彩，相关 issue [#1904](https://github.com/xuri/excelize/issues/1904)
* 添加图表时支持为东亚文字与复杂文字脚本設定字体
* 添加图表时支持为面积图和折线图設定下垂线与高低线
* 获取图片时支持返回部分图片格式属性，相关 issue [#2157](https://github.com/xuri/excelize/issues/2157)
* 新增流式設定栏可见性功能，相关 issue [#2075](https://github.com/xuri/excelize/issues/2075)
* 新增流式設定栏的分级显示功能，相关 issue [#2212](https://github.com/xuri/excelize/issues/2212)
* 添加圖形和添加交叉分析篩選器时，支持将圖形与交叉分析篩選器設定为单一存储格锚定类型
* 获取交叉分析篩選器时支持获取单一存储格锚定类型的交叉分析篩選器
* 支持設定与获取“3 个三角形”、“3 颗星”和“5 个方框”类型的图标集條件式格式，相关 issue [#2038](https://github.com/xuri/excelize/issues/2038)
* 删除條件式格式和删除资料验证时，支持从一个大的存储格范围中删除指定存储格范围内的條件式格式或资料验证，而保留其余存储格范围内的條件式格式或资料验证
* 添加图片时支持設定图片的名称
* 添加图表或插入圖形时，支持为图表或圖形設定名称与替代文本
* 添加交叉分析篩選器时支持为交叉分析篩選器設定替代文本
* 支持校验图形名称与替代文本的长度，当长度超过限制时将会返回错误
* 支持以 UTF-16 方式检查并截断文本长度

### 兼容性提升

* 保存活页簿时将自动移除无效的空列，减少生成的活页簿文件大小

### 问题修复

* 修复 v2.10.0 中引入的问题，解决 `GetCellValue` 与 `GetRows` 函式在某些情况下读取空白存储格时，错误地返回了共享字符串索引，相关 issue [#2240](https://github.com/xuri/excelize/issues/2240)
* 修复 `GetPivotTables` 函式在部分情况下获取枢纽分析表时发生 panic 的问题
* 修复部分情况下，读取带有中文月份名称数字格式的存储格时，发生 panic 的问题，相关 issue [#2224](https://github.com/xuri/excelize/issues/2224)
* 修复部分情况下，打开带有密码保护的加密活页簿时，发生 panic 的问题，相关 issue [#2237](https://github.com/xuri/excelize/issues/2237)
* 修复使用流式写入器 `SetRow` 函式时，栏样式缺失的问题
* 修复获取图片时，未包含部分存储格图片的问题
* 修复由浅色主题色彩索引溢出导致的，部分情况下生成的活页簿损坏的问题
* 修复删除资料验证函式 `DeleteDataValidation` 在资料验证存储格范围无序时，资料验证存储格范围未被正确更新的问题
* 修复設定條件式格式函式 `SetConditionalFormat` 在設定部分带有时间周期條件式格式规则时，生成的活页簿损坏的问题
* 修复计算存储格的值时，对带有单引号的工作表名称解析失败，导致的存储格公式计算结果有误问题
* 修复使用默认字体或填充格式创建样式时，重复创建样式的问题，相关 issue [#2254](https://github.com/xuri/excelize/issues/2254)

### 性能优化

* 透过增加计算缓存并将处理范围限定到实际资料区域，优化了公式计算引擎的性能，相关 issues [#2057](https://github.com/xuri/excelize/issues/2057) 和 [#2223](https://github.com/xuri/excelize/issues/2223)
* 优化算带有 `VLOOKUP` 函式的公式计算性能，内存分配与耗时最多降低约 50%，相关 issue [#2139](https://github.com/xuri/excelize/issues/2139)
* 优化了合并存储格重叠范围的检查算法，降低了获取合并存储格函式 `GetMergeCells` 的内存分配和耗时，相关 issue [#2226](https://github.com/xuri/excelize/issues/2226)
* 透过使用连分数基本递推公式转换优化了带有分数数字格式代码的存储格读取速度

### 其他

* Go Modules 依賴模塊更新
* 單元測試與文檔更新
* 包含阿拉伯語、德語、英語、西班牙語、法語、意大利語、日語、韓語、葡萄牙語、俄語、簡體中文和繁體中文的[多國語言文檔網站](https://xuri.me/excelize)更新
* 支援 WebAssembly / JavaScript 的 [excelize-wasm](https://github.com/xuri/excelize-wasm) NPM 包發佈版本更新
* 支援 Python 的 [excelize](https://github.com/xuri/excelize-py) PyPI 包發佈版本更新
* 支援 C# 的 [ExcelizeCs](https://github.com/xuri/excelize-cs) NuGet .Net 包發佈
