CultureName 是受支援的區域格式類別定義。

```go
type CultureName byte
```

部分儲存格數字格式受到作業系統特定的區域日期和時間設定的影響，以下這些常量定義了當前支援的區域格式類別列舉：

```go
const (
    CultureNameUnknown CultureName = iota
    CultureNameEnUS
    CultureNameJaJP
    CultureNameKoKR
    CultureNameZhCN
    CultureNameZhTW
)
```
